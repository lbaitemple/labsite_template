# Images Directory

Place your images in the appropriate subdirectories:

## Directory Structure

- **team/** - Current team member photos
  - Use format: firstname-lastname.jpg
  - Recommended size: 200x200px to 400x400px
  
- **team/alumni/YEAR/** - Alumni photos organized by graduation year
  - Create folders for each year (e.g., 2023/, 2024/)
  - Use same naming convention as current members

- **slider/** - Homepage carousel images
  - All images in this folder will automatically appear in the slider
  - Recommended size: 1200x400px (will be cropped to 400px height)
  - Supported formats: .jpg, .png, .gif

- **research/** - Research project images
  - Name descriptively (e.g., project-name-diagram.png)

- **news/** - News article images
  - Referenced in _data/news.yml
  - Recommended size: 400x300px

- **logo/** - Lab logo and branding
  - Used in header/footer

## Image Guidelines

- **Format**: Use .jpg for photos, .png for graphics with transparency
- **Size**: Optimize images before uploading (use tools like TinyPNG)
- **Naming**: Use lowercase with hyphens, no spaces
- **Quality**: Balance quality and file size (aim for < 500KB per image)

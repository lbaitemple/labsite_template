ACADEMIC LAB WEBSITE TEMPLATE
==============================

This template includes everything you need to create a professional academic lab website.

-------------------

├── _data/                   # Data files
│   ├── news.yml            # News items

│   └── tweets.html

│   │   └── sample-project.md

✓ images/README.md - Image guidelines

READY TO USE
------------
1. Edit _config.yml with your lab info
2. Replace sample content with your own
3. Add your images
4. Run: bundle exec jekyll serve
5. Deploy to GitHub Pages

For detailed instructions, see QUICKSTART.md or SETUP.md
